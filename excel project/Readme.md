# 🏪 Decathlon Retail Sales Dashboard (Microsoft Excel)

## 📌 Project Overview

This project is an interactive Microsoft Excel dashboard developed to analyze the retail sales performance of a Decathlon-style sporting goods retail chain.

The dashboard transforms raw transactional data into meaningful business insights using Pivot Tables, Pivot Charts, KPI Cards, Slicers, and Excel formulas.

---

## 🎯 Business Objective

The objective of this dashboard is to help management monitor business performance and support data-driven decision-making by analyzing:

- Sales Performance
- Profitability
- Store Performance
- Product Performance
- Customer Behavior
- Payment Methods
- Customer Ratings
