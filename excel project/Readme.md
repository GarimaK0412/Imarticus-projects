# 🏪 Decathlon Retail Sales Dashboard

An interactive Microsoft Excel dashboard developed to analyze the retail performance of a Decathlon-style sporting goods retail chain.

The dashboard transforms raw transactional sales data into meaningful business insights using Pivot Tables, Pivot Charts, KPI Cards, Slicers, and Excel formulas.

# 🎯 Business Objective

The objective of this dashboard is to help management monitor retail performance and support business decision-making through interactive visualizations.

The dashboard provides insights into:

- Sales Performance
- Profitability
- Product Performance
- Store Performance
- Customer Behaviour
- Regional Sales
- Payment Methods
- Customer Satisfaction

---

# 📁 Dataset

The dataset contains approximately **1,500 retail transactions** from a simulated Decathlon retail chain.

# 📌 Key Insights

- Monthly sales trends help identify seasonal demand.
- Running and Cycling products generate significant revenue.
- Regional analysis highlights top-performing markets.
- Brand-wise sales comparison assists inventory planning.
- Customer ratings provide valuable feedback on shopping experience.
- Interactive slicers enable dynamic filtering across the dashboard.
- KPI cards provide an executive overview of business performance.

---

# 💡 Business Recommendations

- Increase inventory for high-performing product categories.
- Review discount strategies to improve profit margins.
- Replicate best practices from top-performing stores.
- Promote loyalty memberships to improve customer retention.
- Focus marketing efforts on underperforming regions.
- Continuously monitor customer satisfaction to improve service quality.
